## 強調
```
#斜体(*,_)
通常 *斜体* 
通常 _斜体_

#太字(**,__)
通常 **太字** 
通常 __太字__

#複合(***,___)
通常 ***複合*** 
通常 ___複合___
```
通常 *強調*  
通常 _強調_  
通常 **太字**  
通常 __太字__  
通常 ***複合***  
通常 ___複合___

# Mermaidによるクラス図作成

## グラフ
グラフの向きはTB、TD、BT、RL、LRなどでどの方向からどの向きに書くかをコントロール可能<br>
⇔を-に変えることで無向グラフも可能
```
$```mermaid
graph RL;
  A-->B;
  A---C;
  B-->D;
  C-->D;
$```
```

```mermaid
flowchart LR;
  A-->B;
  A---C;
  B-->D;
  C-->D;
```
